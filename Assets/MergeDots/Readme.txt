___Levels___
The game only has 31 levels but you can add/edit/delete levels.

For adding your own levels, you can follow these steps:
1. Open Game scene
2. Duplicate Spot prefabs and move them in grid using Ctrl key
3. Click on "GameController" object in the Hierarchy and then tick "Get Code" (in Inspector)
4. Run the game, then copy the Generated code (in Inspector)
5. Stop running 
6. Paste the code in Levels array in Inspector (element number is level number)


___Admob Ads___
The game only has Interstitial ads and it runs every 3 levels (after level complete)
You can go to Scripts > Admob.cs  and then change the IDs to your own.
